# Materia-POO

Sistema de Compra de Camisas
Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema de e-commerce para venda de camisas personalizadas. A aplicação permite que usuários visualizem produtos, personalizem estampas e realizem compras de forma prática e segura.

Objetivo

Criar uma plataforma de venda de camisas com foco em:

Experiência do usuário
Personalização de produtos
Facilidade no processo de compra
Funcionalidades
Catálogo de Produtos
Listagem por categorias: masculina, feminina e infantil
Filtros por tamanho, cor, preço e marca
Exibição de imagens, descrição e avaliações
Personalização de Camisas
Escolha de cor
Inserção de texto personalizado
Upload de imagens ou logos
Pré-visualização da camisa
Carrinho de Compras
Adicionar e remover produtos
Alterar quantidade
Aplicação de cupons de desconto
Cálculo automático do total
Checkout
Cálculo de frete por CEP
Seleção de endereço de entrega
Formas de pagamento: cartão, boleto e Pix
Resumo do pedido
Gestão de Pedidos
Histórico de compras
Acompanhamento de status
Detalhes do pedido
Fluxo da Aplicação
Usuário acessa o catálogo
Visualiza um produto
Personaliza (opcional)
Adiciona ao carrinho
Finaliza a compra
Acompanha o pedido
Funcionalidades Futuras
Sistema de login e cadastro
Perfil do usuário
Gerenciamento de endereços
Painel administrativo
Controle de estoque
